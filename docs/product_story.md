# Product Story — PromptLab

PromptLab 是一个面向普通用户与专业团队的 Prompt 写作助手。它的核心使命是让用户从“会用 AI”向“会写 Prompt”升级。

## 产品情景
对于多数 AI 用户而言，真正困难的不是获取 AI，而是告诉 AI 做什么。一个模糊的请求会导致无关、低效甚至错误的输出。PromptLab 通过评分、诊断、优化与模板，帮助用户构建高质量 Prompt，提升输出可控性与效率。

## 用户价值
- 降低 Prompt 试错成本
- 让非技术用户也能输出专业级 Prompt
- 让团队共享最佳 Prompt 模板
- 让 Prompt 变成可度量、可复用的资产

## 产品故事要点
1. 发现问题：用户写 Prompt 难、结果差、无法复盘
2. 提供方案：评分+诊断+优化+模板+历史记录
3. 核心价值：从“写不懂”到“写得好”、从“经验”到“可复用”
4. 未来形态：个人网页工具 + 团队协作版 + API 服务

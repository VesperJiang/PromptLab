# PromptLab：让 Prompt 质量看得见、改得准
- AI 效果的开始，是优质 Prompt
- 我们将 Prompt 写作从经验转化为可评估能力
- 目标：让用户快速写出高质量、可复用的 Prompt

Note: 开场明确产品价值与业务定位。

---

# 当前痛点
- 多数用户不会写出高质量 Prompt
- AI 输出常常结果不稳定、需要重复调整
- 现有工具侧重“执行”，缺少“写作质量反馈”
- 这是 AI 生产力提升的关键缺口

Note: 强调问题的规模与紧迫性。

---

# 目标用户与市场机会
- 学生与教育：写作辅助、作业生成、学习复盘
- 内容创作者：文案、营销、短视频脚本
- 企业团队：客服、销售、产品、运营文档
- 机会：Prompt 已成为所有 AI 场景的入口

Note: 展示对用户群体和商业机会的覆盖。

---

# PromptLab 核心价值
- 评分体系：清晰度、具体性、约束、上下文
- 诊断反馈：直接指出写作短板
- 智能优化：生成更优 Prompt 候选
- 模板与历史：支持快速复用与结果可追溯

Note: 说明我们的产品核心如何解决痛点。

---

# 产品演示流程
1. 输入原始 Prompt
2. 展示分项评分与诊断结果
3. 生成 2-3 条优化候选
4. 用户选择并保存最佳版本
5. 展示改进后可视化结果

Note: 用简洁步骤说明 Demo 的呈现方式。

# 关键案例演示
- 原始 Prompt："写一封产品发布邮件"
- 评分：60/100，问题：目标不明确、受众不清
- 优化结果：明确受众、加入 CTA、规范格式
- 改进后：评分 85/100，输出更稳定、效果更好

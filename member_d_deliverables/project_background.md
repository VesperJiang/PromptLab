# Project Background — PromptLab

## 项目背景

随着大规模语言模型（LLM）和生成式 AI 工具的普及，普通用户越来越依赖 ChatGPT、DeepSeek、Gemini 等服务完成写作、编程、学习和办公任务。然而，大多数用户并没有掌握“Prompt Engineering”的最佳实践。写法不当的 Prompt 会导致结果不准确、不可复用或不可评估，从而降低 AI 的实际价值。

PromptLab 旨在成为一个面向大众与专业用户的 Prompt 工具箱，帮助用户分析、评分、优化并管理 Prompt，提升 AI 应用的产出质量。

## Problem Statement（问题陈述）

1. 许多用户不会写清晰、可执行的 Prompt，导致 AI 输出不符合预期。
2. 用户无法量化 Prompt 的质量，也不知道如何改进。
3. 缺乏便捷的历史记录与版本管理来比较不同 Prompt 的效果。
4. 现有工具难以为不同场景提供可复用的模板与示例。

## 业务价值

- 提高 AI 输出质量，节省用户试错时间
- 为教育、企业培训和内容创作者提供可重复的 Prompt 模板
- 通过高级模板库或团队版本实现商业化变现（订阅 / 团队版）

## 关键目标（OKR）

- O: 在 6 周内完成可演示的 PromptLab 原型并进行课堂/群体测试
  - KR1: 收集 30—50 份用户问卷并完成初步分析
  - KR2: 实现 Prompt 诊断、评分与自动优化三大核心功能并在 Demo 中演示
  - KR3: 制作不少于 10 个高质量场景模板并集成模板库
  - KR4: 完成市场推广计划并设计产品落地方案（网页 + 插件）

## 关键指标（建议）

- 用户满意度（问卷评分）
- 优化后 Prompt 有效率提升（成功率/质量评分）
- 模板使用率
- 转换率（使用免费版后升级高级模板库）

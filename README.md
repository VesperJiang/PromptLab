# PromptLab: AI Prompt Engineering Studio

PromptLab is a web-based tool that helps users diagnose, score, optimize, and manage prompts for large language models.

## Project Goal

Many users get unsatisfactory AI responses because their prompts are vague, incomplete, or lack clear requirements. PromptLab aims to help users write better prompts by providing prompt diagnosis, prompt scoring, prompt optimization, and prompt templates.

## Core Functions

1. Prompt Diagnosis  
   Check whether a prompt includes role, task, context, constraints, and output format.

2. Prompt Score  
   Evaluate prompt quality from several dimensions, including clarity, completeness, context, constraints, and output format.

3. Prompt Optimization  
   Generate an improved version of the original prompt and explain the changes.

4. Prompt Template Library  
   Provide reusable prompt templates for studying, coding, writing, office work, and content creation.

5. Prompt History  
   Save prompt records locally for later review.
